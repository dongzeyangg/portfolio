Guide simplifié

Installer Debian 12

Bookworm — De l'ISO au bureau en 6 étapes.

Debian 12 "Bookworm" — LTS · Juin 2023
01

Prérequis matérielsHardware

  • 🖥️Processeur : x86-64 (AMD64) recommandé
  • 💾RAM : 512 MB minimum — 2 GB recommandés avec interface graphique
  • 💿Disque : 10 GB minimum — 20 GB+ recommandés
  • 🌐Réseau : conseillé pour télécharger les paquets durant l'installation
  • 🔌Clé USB : 4 GB minimum pour l'image ISO
💡

Debian est très léger. Il peut tourner sur des machines avec seulement 512 MB de RAM en mode serveur (sans interface graphique).

02

Télécharger l'ISO & créer la clé USBDownload & Media

  1. 1
    Téléchargez l'ISO netinst (petite, ~400 MB) ou DVD (complète, ~4 GB) sur debian.org/distrib.
  2. 2
    Téléchargez Rufus sur rufus.ie et branchez votre clé USB.
  3. 3
    Dans Rufus, sélectionnez l'ISO Debian, schéma GPT (UEFI), puis cliquez sur Démarrer.
⚠️

Avec l'ISO netinst, une connexion internet filaire est nécessaire pendant l'installation pour télécharger les paquets.

03

Démarrer l'installateurBoot & Setup

  1. 1
    Redémarrez avec la clé USB branchée et accédez au menu de boot (F12, F11 ou Esc).
  2. 2
    Choisissez Graphical Install pour une installation guidée avec interface graphique.
  3. 3
    Sélectionnez la langue (Français), votre pays et la disposition du clavier.
💡

Choisissez Install (mode texte) si votre machine est très ancienne ou sans affichage graphique.

04

Réseau, compte & partitionsNetwork, Users & Disk

  1. 1
    Entrez le nom de la machine (ex : debian-pc) et laissez le domaine vide si pas en réseau d'entreprise.
  2. 2
    Définissez le mot de passe root (administrateur système) — ou laissez vide pour désactiver root et utiliser sudo.
  3. 3
    Créez votre compte utilisateur (nom complet, identifiant, mot de passe).
  4. 4
    Partitionnement : choisissez Guidé — utiliser tout le disque pour une installation simple.
  5. 5
    Confirmez l'écriture des partitions sur le disque et patientez.
🗑️

Le partitionnement efface toutes les données du disque sélectionné. Sauvegardez vos fichiers importants avant.

05

06

Fin d'installation & premier démarrageFirst Boot

  1. 1
    L'installateur installe GRUB (le chargeur de démarrage) — laissez-le sur le disque principal.
  2. 2
    Retirez la clé USB quand demandé, puis confirmez le redémarrage.
  3. 3
    Connectez-vous avec votre compte utilisateur créé pendant l'installation.
  4. 4
    Mettez à jour le système avec les commandes ci-dessous.
Terminal # Passer en super-utilisateur $su - # Mettre à jour la liste des paquets et le système #apt update && apt upgrade -y # Installer sudo (si non présent) et ajouter l'utilisateur #apt install sudo -y #usermod -aG sudo votre_utilisateur
✅

Debian 12 est installé et à jour ! Vous pouvez maintenant installer vos logiciels avec apt install nom-du-paquet.
