Guide simplifié

Windows Server 2022

Installation pas à pas en 5 étapes clés.

01

Prérequis matérielsHardware

  • 🖥️Processeur : 1.4 GHz 64-bit minimum
  • 💾RAM : 512 MB minimum — 4 GB recommandés
  • 💿Disque : 32 GB minimum
  • 🌐Réseau : carte Ethernet (Gigabit recommandé)
  • 🔒Firmware : UEFI + TPM 2.0 recommandés
02

Télécharger l'ISODownload

  • 🆓Évaluation 180 jours — sur microsoft.com/evalcenter, sans clé requise.
  • 🔑Licence production — via le portail VLSC ou votre revendeur Microsoft.
ℹ️

L'ISO fait environ 5 GB. Gardez le fichier téléchargé pour créer la clé USB bootable.

03

Créer la clé USB bootableBootable Media

  1. 1
    Téléchargez Rufus sur rufus.ie (gratuit).
  2. 2
    Branchez une clé USB de 8 GB+ et lancez Rufus.
  3. 3
    Sélectionnez votre ISO, choisissez le schéma GPT (UEFI), puis cliquez sur Démarrer.
⚠️

La clé USB sera entièrement effacée. Sauvegardez son contenu avant.

04

BIOS & InstallationSetup

  1. 1
    Redémarrez et accédez au BIOS (F2, Del ou F12 selon la marque). Placez la clé USB en premier dans l'ordre de démarrage.
  2. 2
    Au démarrage, sélectionnez la langue et cliquez sur Installer maintenant.
  3. 3
    Choisissez l'édition : Standard (Desktop Experience) pour l'interface graphique.
  4. 4
    Sélectionnez Installation personnalisée, choisissez le disque et laissez l'installation se terminer (~20 min).
⚠️

N'éteignez pas le serveur pendant l'installation. Il redémarre plusieurs fois automatiquement.

05

Configuration initialeFirst Boot

  1. 1
    Définissez un mot de passe Administrateur fort au premier démarrage.
  2. 2
    Renommez le serveur via le Gestionnaire de serveur → Serveur local → Nom de l'ordinateur.
  3. 3
    Attribuez une adresse IP fixe dans les paramètres réseau.
  4. 4
    Lancez Windows Update et installez toutes les mises à jour disponibles.
  5. 5
    Redémarrez — votre serveur est opérationnel !
✅

Installation terminée. Vous pouvez maintenant installer des rôles (AD DS, IIS, Hyper-V…) via le Gestionnaire de serveur.
