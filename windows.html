Installer Windows 10 & 11

Choisissez votre version et suivez les étapes.

Windows 10
Windows 11
Différences
Windows 10 — Supporté jusqu'en octobre 2025
01

Prérequis matérielsHardware

  • 🖥️Processeur : 1 GHz minimum
  • 💾RAM : 1 GB (32-bit) ou 2 GB (64-bit)
  • 💿Disque : 16 GB (32-bit) ou 20 GB (64-bit)
  • 🎨Affichage : 800×600 minimum
ℹ️

Windows 10 est très tolérant avec le matériel ancien. Il tourne sur la plupart des PC depuis 2010.

02

Créer la clé USB bootableBootable Media

  1. 1
    Téléchargez le Media Creation Tool sur microsoft.com/fr-fr/software-download/windows10.
  2. 2
    Lancez-le et choisissez Créer un support d'installation.
  3. 3
    Sélectionnez Clé USB, choisissez votre clé (8 GB+) et laissez le téléchargement se terminer.
⚠️

La clé USB sera entièrement effacée. Sauvegardez son contenu avant.

03

Démarrer sur la clé USBBoot

  1. 1
    Redémarrez le PC avec la clé USB branchée.
  2. 2
    Appuyez sur F12, F11 ou Esc au démarrage pour ouvrir le menu de boot.
  3. 3
    Sélectionnez votre clé USB dans la liste et appuyez sur Entrée.
💡

La touche de boot varie selon la marque : Dell F12 · HP F9 · Asus F8 · Lenovo F12.

04

Installer Windows 10Setup

  1. 1
    Sélectionnez la langue et le clavier, puis cliquez sur Installer maintenant.
  2. 2
    Entrez votre clé de produit ou cliquez sur Je n'ai pas de clé.
  3. 3
    Choisissez l'édition : Windows 10 Famille ou Professionnel.
  4. 4
    Acceptez le contrat, puis sélectionnez Installation personnalisée.
  5. 5
    Choisissez le disque de destination et cliquez sur Suivant. L'installation démarre (~20 min).
⚠️

Le PC redémarre plusieurs fois automatiquement. Ne débranchez pas la clé USB avant la fin.

05

Configuration initiale (OOBE)First Boot

  1. 1
    Choisissez votre région et la disposition du clavier.
  2. 2
    Connectez-vous à votre réseau Wi-Fi (ou passez cette étape).
  3. 3
    Choisissez Configurer pour un usage personnel et connectez-vous avec un compte Microsoft ou créez un compte local.
  4. 4
    Configurez le code PIN et les options de confidentialité selon vos préférences.
✅

Windows 10 est installé ! Lancez Windows Update pour installer les dernières mises à jour.

Windows 11 — Version actuelle recommandée
01

Prérequis matérielsHardware

  • 🖥️Processeur : 1 GHz, 2 cœurs, 64-bit — compatible liste Intel/AMD
  • 💾RAM : 4 GB minimum
  • 💿Disque : 64 GB minimum
  • 🔒TPM 2.0 obligatoire
  • 🛡️Secure Boot activé dans le BIOS
  • 🎨Affichage : 720p minimum, 9 pouces+
⚠️

Windows 11 exige le TPM 2.0. Vérifiez sa présence avec tpm.msc avant de commencer.

02

Vérifier la compatibilitéPC Health Check

  1. 1
    Téléchargez l'outil PC Health Check sur le site de Microsoft.
  2. 2
    Lancez-le et cliquez sur Vérifier maintenant.
  3. 3
    Si votre PC est compatible, passez à l'étape suivante. Sinon, vérifiez le TPM et le Secure Boot dans le BIOS.
💡

Pour activer le TPM : BIOS → onglet Sécurité → TPM Device ou PTT → Activé.

03

Créer la clé USB bootableBootable Media

  1. 1
    Téléchargez le Media Creation Tool Windows 11 sur microsoft.com/fr-fr/software-download/windows11.
  2. 2
    Lancez-le, choisissez Créer un support d'installation et sélectionnez Clé USB (8 GB+).
  3. 3
    Attendez la fin du téléchargement et de la création (~10–15 min).
⚠️

La clé USB sera entièrement effacée. Sauvegardez son contenu avant.

04

Démarrer & InstallerSetup

  1. 1
    Redémarrez avec la clé USB branchée et accédez au menu de boot (F12 / F11 selon la marque).
  2. 2
    Sélectionnez la langue, puis cliquez sur Installer maintenant.
  3. 3
    Entrez votre clé de produit ou cliquez sur Je n'ai pas de clé.
  4. 4
    Choisissez l'édition (Famille ou Professionnel), acceptez le contrat.
  5. 5
    Sélectionnez Installation personnalisée, choisissez le disque, puis laissez l'installation se terminer (~25 min).
🗑️

Une installation propre efface le disque. Sauvegardez vos données importantes avant de continuer.

05

Configuration initiale (OOBE)First Boot

  1. 1
    Choisissez votre pays et la disposition du clavier.
  2. 2
    Connectez-vous au Wi-Fi — Windows 11 Famille demande une connexion internet obligatoire.
  3. 3
    Connectez-vous avec votre compte Microsoft ou créez-en un.
  4. 4
    Configurez le code PIN, choisissez vos options de confidentialité et attendez le bureau.
💡

Pour créer un compte local sans compte Microsoft : déconnectez le Wi-Fi avant l'étape de connexion, puis choisissez Options de connexion → Compte hors connexion.

✅

Windows 11 est installé ! Lancez Windows Update pour récupérer les dernières mises à jour.

📊

Windows 10 vs Windows 11Comparison

WIN 10

  • Supporte le matériel ancien (depuis 2010)
  • Pas de TPM requis
  • Menu Démarrer à gauche classique
  • Fin de support : octobre 2025
  • RAM minimum : 1 GB
  • Stockage minimum : 16 GB
  • Compte local possible facilement

WIN 11

  • Processeurs récents requis (2017+)
  • TPM 2.0 obligatoire
  • Menu Démarrer centré, nouvelle interface
  • Support long terme (2031+)
  • RAM minimum : 4 GB
  • Stockage minimum : 64 GB
  • Compte Microsoft quasi-obligatoire
💡

Vous avez un PC récent (2017+) ? Choisissez Windows 11 pour le support long terme. PC ancien ? Restez sur Windows 10 en attendant un remplacement.

🔑

Clé de produitLicensing

  • 🆓Mise à niveau gratuite — depuis Windows 10 vers Windows 11 via Windows Update si le PC est compatible.
  • 🔑Achat — Famille (~145 €) ou Professionnel (~260 €) sur le Microsoft Store.
  • 🎓Étudiant / entreprise — licences disponibles via votre école ou votre employeur.
⚠️

Sans clé, Windows fonctionne mais affiche un filigrane et certaines options de personnalisation sont bloquées.
